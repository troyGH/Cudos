About ShoutMS

Employee Review Service

Created at San Jose State University, ShoutMS is an application that allows users to rate an employees of any businesses. We believe it's unfair for businesses' ratings impacted by an employee's performance. Instead, employees should have a public profile for others to rate. Not only an employee's rating won't affect the business, employers can track an employee's performance based on the ratings. Customers no longer need to direct to business's directory for customer service forms & surveys, instead, they can be taken more seriously by submitting a review for the public and employers to see. Potential customers can now read reviews of an employee before choosing his or her service.

How To Write a Review

Simple & Similar to Most Review Services

Customers

User must have an account with us via Facebook, Gmail, or Twitter.

User geographically searches for a business by entering a business name and location.

User views the list of employees employed for the selected business.

User selects an employee. If not found, user select "Create New".

User views employee profile and previous reviews.

User clicks on the number of stars and write a description of his/her experience.
